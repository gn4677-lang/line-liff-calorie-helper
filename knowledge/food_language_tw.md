# 台灣飲食口語理解

## Goal

讓 LLM 與 parser 更懂台灣人怎麼講吃了什麼、吃多少、以及在問什麼。

## 常見描述模式

- 餐別 + 食物類型
  - 例：早餐蛋餅、午餐便當、晚餐牛肉麵
- 食物 + 飲料
  - 例：雞腿便當加奶茶
- 份量比例
  - 例：飯半格、吃七分滿、剩三分之一
- 配料與調整
  - 例：少飯、去皮、無糖、半糖、少冰、加珍珠
- 情緒式描述
  - 例：破戒、吃爆、罪惡、救我

## 高頻模糊詞

- 一碗
- 一份
- 一點點
- 幾口
- 差不多
- 正常
- 少一點
- 吃一半
- 沒有很多

## 常見別名與錯字

- 魯肉飯 -> 滷肉飯
- 爌肉飯 -> 控肉飯
- 珍奶 -> 珍珠奶茶
- 黑咖 / 黑咖啡 -> 美式咖啡
- 無糖豆 -> 無糖豆漿
- cc / c.c. -> ml

## 解析優先欄位

- 餐別
- 主食
- 主菜
- 飲料
- 甜度 / 冰量 / 配料
- 飯量 / 份量比例
- 烹調方式
- 是否吃完

## 對話含義提醒

- 「幫我估一下」通常是在求熱量估算。
- 「這樣會不會爆」通常是在求風險判斷與替代方案。
- 「我又破戒了」不一定只是在求熱量，也可能是在求安撫或補救策略。
- 「今天都外食」常代表使用者需要的是整天規劃，而不只是單餐估算。
