## 1. 設計系統 Design System

### 色彩方案

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg-primary` | `#0f0f1a` | 最深背景&#44; 頁面底色 |
| `--bg-card` | `#1a1a2e` | 卡片背景 |
| `--bg-card-elevated` | `#222240` | 浮起的卡片（draft card&#44; overlay） |
| `--accent-warm` | `#f0a030` | 主要 accent&#44; CTA 按鈕&#44; 進度環 |
| `--accent-warm-glow` | `rgba(240,160,48,0.15)` | accent 光暈效果 |
| `--text-primary` | `#f0f0f5` | 主文字 |
| `--text-secondary` | `#8888aa` | 次要說明文字 |
| `--text-muted` | `#555570` | 佔位文字、分隔線文字 |
| `--success` | `#34d399` | 正常軌道、已確認 |
| `--warning` | `#fbbf24` | 略超、粗估 badge |
| `--danger` | `#f87171` | 明顯超標 |
| `--surface-glass` | `rgba(255,255,255,0.04)` | glassmorphism 表面 |

> [!IMPORTANT]
> **拒絕紫色 AI 預設風格。** 用暖色調暗黑主題，讓體驗像私人教練的工具，不是另一個科技 demo。

### 字型

- **主字型：** `Inter` (Latin) + `Noto Sans TC` (中文)
- **數字強調：** `DM Mono` 或 `JetBrains Mono`（用於 kcal 數字）
- **字級：**

| Token | Size | Weight | 用途 |
|-------|------|--------|------|
| `--text-hero` | 32px | 700 | 核心數字（剩餘 kcal） |
| `--text-title` | 20px | 600 | 頁面標題 |
| `--text-heading` | 16px | 600 | 卡片標題 |
| `--text-body` | 14px | 400 | 正文 |
| `--text-caption` | 12px | 400 | 標籤、badge |
| `--text-micro` | 10px | 500 | 極小標註 |

### 間距與圓角

| Token | Value |
|-------|-------|
| `--spacing-xs` | 4px |
| `--spacing-sm` | 8px |
| `--spacing-md` | 16px |
| `--spacing-lg` | 24px |
| `--spacing-xl` | 32px |
| `--radius-sm` | 8px |
| `--radius-md` | 12px |
| `--radius-lg` | 16px |
| `--radius-pill` | 999px |

### 元件風格原則

- **卡片：** `border-radius: var(--radius-lg)`, 背景 `var(--bg-card)`, 1px `rgba(255,255,255,0.06)` border
- **Glassmorphism：** 謹慎使用在浮起元素（active draft card），`backdrop-filter: blur(12px)` + `var(--surface-glass)` bg
- **Answer chips：** pill 形狀，outline 預設，選中填滿 `var(--accent-warm)`
- **CTA 按鈕：** 主要用 `var(--accent-warm)` 填滿 + `border-radius: var(--radius-pill)`，次要用 outline
- **微動畫：** hover/tap 用 `transform: scale(0.97)` + `transition: 150ms ease`，進度環用 `stroke-dashoffset` 動畫

## 3. Page 1：今日紀錄 Today

### 區塊結構與確切內容

#### 3.1 熱量核心區 Hero Section

這是整頁最重要的視覺錨點。

```
┌────────────────────────────────────────┐
│                                        │
│         ╭──────────────────╮           │
│         │    ⟲ 進度環       │           │
│         │                  │           │
│         │   剩餘            │           │
│         │   740            │           │
│         │   kcal           │           │
│         │                  │           │
│         ╰──────────────────╯           │
│                                        │
│   已吃 860          目標 1,600         │
│   ▓▓▓▓▓▓▓▓░░░░░░   kcal              │
│                                        │
│   本週狀態  ● 正常       [補償中 badge] │
│                                        │
└────────────────────────────────────────┘
```

**確切元素：**
| 元素 | 內容 | 設計 |
|------|------|------|
| 進度環 | 圓形 SVG，`consumed / target` 比例填滿 | amber 漸層 stroke，暗底背景環 |
| 環中央數字 | `剩餘 {remaining} kcal` | `--text-hero` 大字，DM Mono |
| 底部明細 | `已吃 {consumed}` + `目標 {target} kcal` | `--text-secondary`，水平排列 |
| 週狀態 badge | `正常` / `略超 ▲` / `超標 ▲▲` | pill badge，對應 success/warning/danger |
| overlay badge | `補償中` (只在 overlay active 時) | 小型 warning pill |

**設計決策：**
- 剩餘熱量是**最強視覺錨點**，不是已吃也不是目標
- 進度環在頁面最頂部，佔 ~25% viewport height
- 數字用 monospace font 讓視覺穩定不跳動

```
│ │ └──────────┘  └──────────────┘    │ │

```

**確切元素：**
| 元素 | 內容 | 設計 |
|------|------|------|
| 卡片頭 | 餐別 emoji + 「{meal_type}草稿」+ 精度模式 tag | heading，右側 pill tag |
| 食物描述 | `parsed_items` 的簡短描述 | body text |
| 熱量主數 | `{kcal_estimate} kcal` | `--text-title` 加粗 |
| 範圍滑桿 | `{kcal_low}` ─●─ `{kcal_high}` | 小型 range indicator |
| 信心條 | 填色進度條 + 文字「高/中等/低」 | success/warning/danger 色 |
| 不確定因素 | `{primary_uncertainties}` | warning 色小字 |
| 追問區塊 | 問題文字 + answer chips | 內嵌卡片，glass bg |
| Answer chips | 3-4 個 pill 按鈕 + 自由輸入 | outline pill，tap 後填滿 |
| 主 CTA | 「確認記錄」 | accent-warm 填滿 pill 按鈕 |
| 次 CTA | 「直接記（用粗估）」 | outline pill 按鈕 |

**設計決策：**
- 草稿卡使用 `bg-card-elevated` + subtle border glow，在視覺上浮起
- 追問區是內嵌子卡片，與食物資訊有視覺層次分離
- Answer chips 水平排列，過長時 wrap 到下一行（不用 horizontal scroll）
- 「我自己說」chip 展開輸入框

## 6. 狀態設計 Empty / Loading / Error

| 狀態 | 設計 |
|------|------|
| **Loading** | skeleton 卡片動畫，不用 spinner |
| **Empty timeline** | 「今天還沒記錄，從這裡開始 →」引導到輸入 |
| **Empty recommendations** | 「先記錄今天吃了什麼，就能看到推薦」|
| **Error** | 溫和措辭：「載入有點問題，幫你重試中」+ retry button |
| **Offline** | 顯示 cached 資料 + 小字「等網路恢復後同步」|

> [!TIP]
> 空狀態和錯誤狀態必須感覺是**被設計過的**，不是剩下來的。

## 7. 微動畫清單

| 動畫 | 觸發 | 效果 |
|------|------|------|
| 進度環填充 | 頁面進入 | `stroke-dashoffset` 從 100% → 當前值，`600ms ease-out` |
| 卡片入場 | 滾動到視區 | `opacity 0→1` + `translateY 12px→0`，`300ms` stagger |
| Chip 選中 | tap | `scale(0.95) → scale(1)` + bg fill，`150ms` |
| 確認成功 | confirm tap | 進度環快閃 + card collapse，`400ms` |
| kcal 數字更新 | 確認/修改後 | 數字 counting animation，`300ms` |
| Tab 切換 | tab tap | page slide left/right，`250ms ease` |
| 草稿消失 | 確認後 | card collapse + fade，reveal timeline entry |
